# 🦷 Odontologia Ideale - Website Institucional

[cite_start]Website institucional responsivo da clínica **Odontologia Ideale**, focada em excelência odontológica, tradição e atendimento humanizado no Belém, SP[cite: 1, 4, 10].

## 🚀 Sobre o Projeto

[cite_start]Este projeto foi desenvolvido para apresentar os serviços da clínica, os profissionais responsáveis e garantir uma presença digital sólida e otimizada para mecanismos de busca (SEO)[cite: 3, 63, 7].

### Principais Especialidades Apresentadas:
* [cite_start]**Ortodontia:** Alinhadores e aparelhos fixos[cite: 19, 25].
* [cite_start]**Implantodontia:** Reposição dentária e carga imediata[cite: 26, 29].
* [cite_start]**Estética Dental:** Clareamento, facetas e lentes de contato[cite: 31, 38, 40].
* [cite_start]**Endodontia:** Tratamento de canal especializado[cite: 56].
* [cite_start]**Periodontia:** Saúde da gengiva e enxertos[cite: 58, 61].

## 🛠️ Tecnologias Utilizadas

* [cite_start]**HTML5:** Estrutura semântica do site[cite: 3].
* [cite_start]**Tailwind CSS:** Framework utilitário para um design moderno e responsivo[cite: 3].
* [cite_start]**JavaScript:** Interações de UI (menus, botões de topo, LGPD)[cite: 3].
* **FontAwesome:** Ícones vetoriais.

## 📈 Otimizações Realizadas

* [cite_start]**SEO Avançado:** Configuração de meta tags, Open Graph para redes sociais, `sitemap.xml` e `robots.txt`[cite: 3, 7, 8].
* [cite_start]**Acessibilidade:** Uso de atributos ARIA e contraste de cores adequado[cite: 3].
* [cite_start]**Conformidade LGPD:** Banner de consentimento de cookies e políticas de privacidade integradas[cite: 3, 4].
* [cite_start]**Performance:** Carregamento otimizado de fontes e scripts[cite: 3].

## 📁 Estrutura de Arquivos Principal

* [cite_start]`index.html`: Página principal com todos os serviços[cite: 3].
* [cite_start]`privacidade.html`: Política de Privacidade em conformidade com a LGPD[cite: 4].
* [cite_start]`termos.html`: Termos de uso do website[cite: 5].
* [cite_start]`sitemap.xml`: Mapa do site para indexação do Google[cite: 7].
* [cite_start]`robots.txt`: Instruções para robôs de busca[cite: 8].
